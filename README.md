# Sales Data Analysis Project

## Overview
This project analyzes sales data using Python to uncover insights about revenue, profit, and product performance.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Insights
- Charlotte generated the highest revenue
- Electronics is the top-performing category
- Keyboard is the most profitable product

## Features
- Data cleaning (handled currency formatting issues)
- Data aggregation and analysis
- Visualization with bar charts
- Dashboard creation
- Exported visual reports

## Files Included
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
- Dashboard images (.png)
